# 水浒英雄传：五幕战役

仿帝国时代2的 2D 等距即时战略（Godot 4.6），水浒题材**五幕战役**，每关机制各异：

| 幕 | 关卡 | 核心机制 |
|---|---|---|
| 一 | 智取生辰纲 | 潜伏伏击：藏芦苇隐身、白胜下蒙汗药麻翻押运队，趁虚夺宝车 |
| 二 | 江州劫法场 | 限时营救：行刑倒计时，杀刽子手暂停，救宋江戴宗后护送登船 |
| 三 | 三打祝家庄 | 盘陀路迷宫：石秀指路点亮安全道，避死巷伏兵，破庄门灭三杰 |
| 四 | 大破连环马 | 兵种克制：连环马成排冲锋，钩镰枪手专砍马腿、诱敌入芦苇陷地 |
| 五 | 梁山泊保卫战 | 据险守城：守聚义厅，沿两条窄堤半渡而击，打退官军四波 |

启动进入**战役地图**选关；所有关卡一开始即可选。

## 遭遇战 · 自由模式（仿帝国时代经营 + 仿魔兽英雄）

菜单底部两种自由模式：**⚔ 遭遇战 · 据守**（防守波次）与 **🤖 AI 对战 (1v1)**（含简单/普通/困难）。一套完整的即时战略经营：

- **资源**：金 + 木两种。喽啰（工人）采金矿/伐木，回聚义厅或仓库卸货，自动循环；右键资源点即采。
- **建造**：选喽啰→命令卡建造菜单（QWER 键帽）→鼠标选址（**半透建筑虚影** + 绿框可建/红框不可）→喽啰施工。可造**兵营**（练兵）、
  **箭楼**（自动放箭防御）、**民居**（加人口上限）、**仓库**（就近卸货）、**集市**（木↔金贸易）。多个喽啰同建更快。
  按住 **Shift** 连续选址 → 建造令排队，工人逐座建过去。
- **续建/拆除（帝国式）**：地基**永不消失**——工人离开后，任意工人**右键工地**即续建（鼠标显示续建光标）。
  **Delete 键**或头像旁 **✕** 按钮拆除选中的己方单位/建筑，**释放占地**防卡位（拆建筑需再按一次确认，Shift+Delete 直接拆）。
- **自动复工**：工人建完/修完会**自动回到原来的采集**（不再傻站工地）。
- **集结点（typed）**：右键空地/资源设**集结点**——设在资源上则记住该资源点，新喽啰**自动去采该类资源**，采空就近补位；无集结点时新喽啰自动去采当前较缺的资源。
  人口上限约束（造民居扩张）。顶栏 **闲置 N** 徽标可点击轮选闲置喽啰，人口满时标红提示。
- **英雄祭坛**：聚义厅训练/复活宋江、林冲、花荣、李逵——每人 **3 主动 + 1 被动**；击杀得经验**升级**，
  每级 1 技能点，在命令卡点 **+** 学习/升级技能（魔兽逻辑，1/3/5 级解锁各级）。
- **Tab 子组**：选一堆人按 **Tab** 在不同兵种/英雄间切换，命令卡只显示当前子组，QWER 对应当前英雄技能。
- **Shift 排队**：按住 Shift 连下多个指令（移动→采集→建造）排队执行。
- **据守模式·官军波次**：点「开战」后官军分 7 波定时来犯（首波 2 分钟倒计时、之后波次叠加）、围攻聚义厅，高太尉压阵。打退全部 = 胜；聚义厅被拆 = 败。
- **AI 对战模式·1v1**：对手是会**自己经营**的电脑官军——随时间积累钱粮、**修箭楼/兵营**、按难度**点兵成军**，攒够一波即倾巢突击你的聚义厅；并会就近**迎战**你的进犯。目标：守住聚义厅，集结兵马**攻破「官军大营」**即胜。难度（简单/普通/困难）决定其进账、出兵频率与每波规模。
- **战争迷雾·记忆**：单位视野较大（兵 8 格 / 建筑 10 格）；单位离开后照亮的区域**延时 30 秒**才变回阴影（不立刻塌黑）。**开过图处的敌方建筑信息会留在阴影里**——你始终记得官军大营/箭楼/兵营的位置（半暗轮廓），但看不到阴影中的敌军调动，须再派兵侦察。
- **作战姿态 / 巡逻 / 待命**：选战斗单位按 **G** 循环 4 种姿态（进攻=追击索敌 / 守备=守阵短追 / 据守=原地只打近敌 / **避战**=不索敌不还手），**P** 巡逻往返迎敌，**S** 原地待命。命令面板显示当前姿态。
- **阵型行军**：一队人受令时排成**朝行军方向旋转的方阵**，**近战在前、远程居中、工人靠后**；整队按**最慢成员速度**推进，不散开。双击单位选中屏幕内**所有同类**。
- **驻军（帝国式 garrison）**：右键己方**箭楼/聚义厅**让单位**进驻**躲避（箭楼 5 人、聚义厅 10 人）；驻入的**远程兵给箭楼增援射箭**；选中建筑点「**出击**」放出驻军；建筑被摧毁时驻军自动弹出、不陪葬。
- **战斗手感**：暴击（英雄25%/兵12%·×1.8）、命中飘**伤害数字**、暴击放大变色、命中火花、受击闪白与退缩、阵亡倒地淡出、施法/暴击/大单位阵亡**屏震**；行进/静止单位**非对称分离**避免站桩抖动。
- **画面氛围（美术 revamp）**：全屏**后期处理**（暗角聚焦 + 暖色分离调 + 对比/饱和提升）、**水面波光**、**地面云隙光斑**、空气**浮尘**、单位**描边**从背景跳出；主菜单亦换上渐变 + 灯笼辉光 + 暗角的新背景。建造时显示**半透建筑虚影**预览。
- **程序合成 BGM**：经营曲（安宁五声）与战斗曲（战鼓行进）随战况无缝交叉切换；连同全部音效均为运行时合成，无需任何音频素材。

命令行直接进：`SKIRMISH=1 …`（据守）或 `SKIRMISH_AI=1 AI_DIFF=hard …`（AI 对战）`/Applications/Godot.app/Contents/MacOS/Godot --path .`

## 运行

1. 打开 Godot 4.6，导入本目录的 `project.godot`；
2. 按 F5 运行（进入战役地图选关）。

命令行运行：

```sh
/Applications/Godot.app/Contents/MacOS/Godot --path .
# 直接进某一关（跳过菜单，调试用）：
LEVEL=3 /Applications/Godot.app/Contents/MacOS/Godot --path .
```

## 玩法与梁山地形

- **八百里水泊**：水面不可通行，官军只能沿两条**窄堤**进攻——咽喉要道，适合长枪手堵口。
- **沼泽泥泞**：梁山兵熟悉水路（沼泽中 85% 移速），官军深陷泥潭（50% 移速），骑兵尤其难受。
- **芦苇荡藏伏兵**：芦苇中未交战的梁山单位处于隐蔽状态，官军靠得很近才会发现——把弓手伏进芦苇，半渡而击。
- **布阵阶段**：开战前可自由调整阵型；点击「开战」后官军分四波来袭，最后一波高俅亲临。
- **胜负**：杀死高俅或打退全部四波 = 胜利；聚义厅被拆或全军覆没 = 失败。
- **主场休整**：梁山兵脱战 6 秒后缓慢回血（官军远征无补给，不回血）——把残兵拉回后排很值得。
- 头领自带光环：宋江（周围攻击+25%）、吴用（周围移速+15%）；第三波时林冲、花荣率伏兵来援。

### 英雄主动技能

选中英雄后，底部面板出现技能按钮（点击施放）。多选英雄时按钮依选择顺序绑定 **Q / W / E / R**，
对应键即可施放各自技能。带 ◎ 的指向技能在**按下技能那一刻**即在鼠标处显示等距作用范围环（随光标移动），
左键确认落点、右键/Esc 取消；指向施法与攻击移动（A）期间鼠标变为「圈中带点」的指向光标：

| 英雄 | 技能 | 冷却 | 效果 |
|---|---|---|---|
| 宋江 | 替天行道 | 24s | 周围梁山兵回血 70、攻击 +60%（8 秒） |
| 吴用 | 锦囊火计 ◎ | 22s | 指定处烈焰，范围内官军受 130 伤害 |
| 林冲 | 丈八横扫 | 15s | 身边官军受 95 伤害并被减速 |
| 花荣 | 百步穿杨 ◎ | 13s | 对目标处官军造成 170 穿透伤害 |

## 美术资源管线

`assets/` 下 7 张原始图集（`*_raw.png`）覆盖全战役角色/地形/头像：
- 第一批：`units_sheet` / `terrain_sheet_raw_iso` / `portraits_sheet`（梁山主将、水泊地形）
- 第二批：`units2`（梁山新好汉）/ `units3`（敌将·特殊·目标）/ `terrain2`（丘陵·城镇·村庄）/ `portraits2`（新登场人物）

运行切图脚本，自动检测网格线精确切格 → 去线 → 单位/物件格泛洪抠底 → 内容居中 → 重组整齐网格，
写出游戏实际加载的 `*.png`：

```sh
/Applications/Godot.app/Contents/MacOS/Godot --headless --path . --script res://tools/process_sheets.gd
```

重画某张原图后重跑该命令即可。图集缺失时游戏自动退回代码占位图形（提示词与网格对照见 `ART_PROMPTS.md`）。

第二批图集由 ChatGPT(DALL·E) 经 Safari 自动化（AppleScript `do JavaScript`）批量生成：
注入提示词 → 发送 → 轮询出图 → 页内 fetch 图片为 dataURL 取回 → 解码存盘。脚本见 `/tmp/cg/`（一次性工具）。

## 动画系统（混合：程序化 + 逐帧接口）

单位默认用单帧立绘 + 丰富的**程序化动画**：
- 行走：两拍步态（每步触地下沉 `cos(2θ)`、落脚 squash／腾空 stretch、重心左右换步、躯干侧倾、随速前倾）、
  脚步朝后蹬尘、腾空时影子收缩；骑兵另加腾空上扬与前后涌动；
- 攻击：**按武器分动作**——刀劈（弧光）、枪刺（刺线）、斧砸（大弧+下劈）、弓张拉撒放；
  伤害在挥击命中瞬间结算（起手预备→命中），命中迸火花；
- 待机：缓慢呼吸；受击：朝反方向退缩一顿 + 闪白；
- 死亡：朝受击反方向倒地 + 淡出（不再瞬间消失）；朝向镜像。

**逐帧接口**：若放入 `assets/anim/<key>_<state>.png`（横向帧带，每帧正方形，state ∈ `idle`/`walk`/`attack`），
该兵种对应状态会自动改播逐帧动画（walk 按步幅相位映射整条带，attack 随挥击进度播一遍，且会自动**压低**
程序化叠加以防腿部重影），否则退回程序化。无需改码。

目前已为 9 个单位生成**真·逐帧走循环**（4 帧）：梁山刀手/枪手、官军刀手/枪手、宋江、林冲、李逵，
以及喽啰（工人）、梁山马军（骑兵，无独立立绘，待机自动取走循环过渡帧，保证全程同一套美术）。
资源点物件（金矿 + 松/阔叶/垂柳三种树）由一张 2×2 绿幕图经 `tools/cut_anim.gd` 切成 `assets/objects.png`
（art_db 以 `object_texture` 取用，林木按节点坐标稳定取一种使林子有变化）。
原图由 ChatGPT 出 2×2 走循环大图（绿幕底），经 `tools/cut_anim.gd` 抠底→去溢色→居中→重排成 1×4 帧带：

```sh
/Applications/Godot.app/Contents/MacOS/Godot --headless --path . --script res://tools/cut_anim.gd
```
