# 水浒英雄传：八幕战役

一款水浒题材的 2D 等距即时战略（Godot 4.6），原创作品。**八幕剧情战役** + **自由经营遭遇战 / 1v1 对战** + **数据驱动的关卡编辑器**。每关机制各异：

| 幕 | 关卡 | 核心机制 |
|---|---|---|
| 一 | 智取生辰纲 | 潜伏伏击：藏芦苇隐身、白胜下蒙汗药麻翻押运队，趁虚夺宝车 |
| 二 | 江州劫法场 | 限时营救：行刑倒计时，杀刽子手暂停，救宋江戴宗后护送登船 |
| 三 | 三打祝家庄 | 盘陀路迷宫：石秀指路点亮安全道，避死巷伏兵，破庄门灭三杰 |
| 四 | 大破连环马 | 兵种克制：连环马成排冲锋，钩镰枪手专砍马腿、诱敌入芦苇陷地 |
| 五 | 梁山泊保卫战 | 据险守城：守聚义厅，沿两条窄堤半渡而击，打退官军四波 |
| 六 | 大闹野猪林 | 救援护送：鲁智深禅杖开路，野猪林中救下被押的林冲 |
| 七 | 醉打蒋门神 | 越喝越勇：武松「无三不过望」沿途畅饮叠加酒力，醉拳破快活林 |
| 八 | 东昌府·飞石 | 飞石点将：没羽箭张清飞石连打，水路设伏擒将招安 |

启动进入**战役地图**选关；所有关卡一开始即可选。

## 遭遇战 · 自由模式（经营养成 + 英雄成长）

菜单底部两种自由模式：**⚔ 遭遇战 · 据守**（防守波次）与 **🤖 AI 对战 (1v1)**（含简单/普通/困难）。一套完整的即时战略经营：

- **资源**：金 + 木两种。喽啰（工人）采金矿/伐木，回聚义厅或仓库卸货，自动循环；右键资源点即采。
- **建造**：选喽啰→命令卡建造菜单（QWER 键帽）→鼠标选址（**半透建筑虚影** + 绿框可建/红框不可，且不得压到别的建筑）→喽啰施工。可造**兵营**（练兵）、
  **箭楼**（自动放箭防御）、**民居**（加人口上限）、**仓库**（就近卸货）、**集市**（木↔金贸易）。多个喽啰同建更快。
  按住 **Shift** 连续选址 → 建造令排队，工人逐座建过去。
- **续建/拆除**：地基**永不消失**——工人离开后，任意工人**右键工地**即续建（鼠标显示续建光标）。
  **Delete 键**或头像旁 **✕** 按钮拆除选中的己方单位/建筑，**释放占地**防卡位（拆建筑需再按一次确认，Shift+Delete 直接拆）。
- **自动复工**：工人建完/修完会**自动回到原来的采集**（不再傻站工地）。
- **集结点（typed）**：右键空地/资源设**集结点**——设在资源上则记住该资源点，新喽啰**自动去采该类资源**，采空就近补位；无集结点时新喽啰自动去采当前较缺的资源。
- **英雄祭坛**：聚义厅训练/复活宋江、林冲、花荣、李逵等——每人 **3 主动 + 1 被动**；击杀得经验**升级**，
  每级 1 技能点，在命令卡点 **+** 学习/升级技能（1/3/5 级解锁各级）。
- **Tab 子组**：选一堆人按 **Tab** 在不同兵种/英雄间切换，命令卡只显示当前子组，QWER 对应当前英雄技能。
- **Shift 排队**：按住 Shift 连下多个指令（移动→采集→建造）排队执行。
- **据守模式·官军波次**：点「开战」后官军分波定时来犯、围攻聚义厅，高太尉压阵。打退全部 = 胜；聚义厅被拆 = 败。
- **AI 对战模式·1v1**：对手是会**自己经营**的电脑官军——积累钱粮、修箭楼/兵营、按难度点兵成军，攒够一波即倾巢突击你的聚义厅，并会就近迎战你的进犯。目标：守住聚义厅，攻破「官军大营」即胜。
- **战争迷雾·记忆**：单位视野较大（兵 8 格 / 建筑 10 格）；离开后照亮区域**延时 30 秒**才变回阴影。开过图处的敌方建筑信息留在阴影里（半暗轮廓），但看不到阴影中的敌军调动。
- **作战姿态 / 巡逻 / 待命**：选战斗单位按 **G** 循环 4 种姿态（进攻 / 守备 / 据守 / 避战），**P** 选择巡逻端点，**S** 原地待命，**H** 临时据守到下一条命令。
- **阵型行军**：默认保持部队相对站位；设置中可切换**松散 / 方阵 / 横列**。方阵与横列会朝行军方向旋转并让近战在前，整队按最慢成员速度推进。双击单位可选中屏幕内所有同类。
- **驻军 garrison**：右键己方**箭楼/聚义厅**让单位**进驻**躲避（箭楼 5 人、聚义厅 10 人）；驻入的远程兵给箭楼增援射箭；选中建筑点「**出击**」放出；建筑被摧毁时驻军自动弹出。
- **战斗手感**：暴击、命中飘**伤害数字**、命中火花、受击闪白与退缩、阵亡倒地淡出、施法/暴击/大单位阵亡**屏震**；非对称分离避免站桩抖动。
- **画面氛围**：全屏**后期处理**（暗角聚焦 + 暖色分离调 + 对比/饱和）、水面波光、地面云隙光斑、空气浮尘、单位描边。
- **程序合成 BGM**：经营曲与战斗曲随战况无缝交叉切换；连同全部音效均为运行时合成，无需音频素材。
- **移动端**：自动切触屏布局——屏上操作栏（攻击/停/姿态/拆/取消）、左下编队、右缘常驻英雄技能轨、英雄托管 AI、长按驻扎、建造拖放定位。

命令行直接进：`SKIRMISH=1 …`（据守）或 `SKIRMISH_AI=1 AI_DIFF=hard …`（AI 对战）`/Applications/Godot.app/Contents/MacOS/Godot --path .`

## 关卡编辑器 · 数据驱动关卡（人人可造关）

游戏内置一套**数据驱动的关卡系统**：一关 = 一份 JSON，由 `scripts/levels/scenario.gd` 解释执行（地图/地形/装饰/布兵/波次/增援/胜负）。

- **可视化编辑器**：主菜单 →「更多 → 🗺 场景编辑器」。俯视网格上刷地形、放兵、放装饰、放波次增援、设出兵口/镜头/开场剧情/胜负预设，保存到本地、一键「▶ 试玩」。做好的关在「更多 → ▶ 玩自定义关卡」里玩。
- **手写 JSON**：完整字段见 [`docs/SCENARIO_FORMAT.md`](docs/SCENARIO_FORMAT.md)，可运行示例见 [`scenarios/`](scenarios/)。
- **内容包（换数值 / 加单位 / 换美术，不改引擎）**：在 `res://content/` 放 `units.json` / `abilities.json`（覆盖或新增）、`content/art/<key>.png`（贴图覆盖）。
- 想基于本引擎做自己的游戏？见 [`MODDING.md`](MODDING.md)。

## 运行

Android 完整 APK、累计 PCK 热更新、签名、服务器部署与回滚流程见
[`docs/ANDROID_RELEASE.md`](docs/ANDROID_RELEASE.md)。

1. 打开 Godot 4.6，导入本目录的 `project.godot`；
2. 按 F5 运行（进入战役地图选关）。

```sh
/Applications/Godot.app/Contents/MacOS/Godot --path .
# 直接进某一关（调试用）：
LEVEL=3 /Applications/Godot.app/Contents/MacOS/Godot --path .
# 跑一份自定义关卡：
SCENARIO=/abs/path/to.json /Applications/Godot.app/Contents/MacOS/Godot --path .
```

## 操作

| 操作 | 按键 |
|---|---|
| 框选 / 点选部队 | 鼠标左键（Shift 加选） |
| 移动 / 攻击 | 鼠标右键（点敌人=攻击） |
| 攻击移动（沿途接敌） | A 后左键点目标（右键/Esc 取消） |
| 待命 / 据守 / 巡逻 / 切姿态 | S 待命 ｜ H 原地据守 ｜ P 后点巡逻端点 ｜ G 循环 4 姿态 |
| 拆除 / 进驻 | Delete（或头像旁 ✕）拆己方 ｜ 右键己方箭楼/聚义厅进驻，选建筑点「出击」放出 |
| 单选 / 选同类 | 点底部面板单位图标 ｜ 双击单位选屏幕内所有同类 |
| 编队 | Ctrl+数字 设队 / Shift+数字 加入 / 数字 选队（双击居中） |
| 视角平移 / 缩放 / 跳转 | 方向键·边缘·中键拖拽 ｜ 滚轮/捏合 ｜ 点小地图 |
| 回到聚义厅 | 空格 |

上述核心命令键位可在“设置 → 核心键位”中重新绑定；按键冲突时两项会自动交换。

触屏：单指拖地图、长按框选、轻点选取/下令、长按建筑驻扎；屏上操作栏与右缘英雄技能轨。

## 玩法与梁山地形

- **八百里水泊**：水面不可通行，官军只能沿窄堤进攻——咽喉要道，适合长枪手堵口。
- **沼泽泥泞**：梁山兵熟悉水路（85% 移速），官军深陷泥潭（50% 移速），骑兵尤其难受。
- **芦苇荡藏伏兵**：芦苇中未交战的梁山单位处于隐蔽状态，官军靠得很近才会发现——弓手伏进芦苇，半渡而击。
- **主场休整**：梁山兵脱战 6 秒后缓慢回血（官军远征无补给，不回血）。
- 头领自带光环：宋江（周围攻击 +25%）、吴用（周围移速 +15%）。

```

scripts/unit.gd        单位状态机 + 动画
scripts/defs.gd        单位/技能/科技/护甲数值表（数据）
scripts/art_db.gd      图集管理（Autoload "Art"）：有图切片用图，无图占位

```

## 美术资源管线

`assets/` 下原始图集（`*_raw.png`）覆盖全战役角色/地形/头像。切图脚本自动检测网格线精确切格 → 去线 → 抠底 → 居中 → 重组：

```sh
/Applications/Godot.app/Contents/MacOS/Godot --headless --path . --script res://tools/process_sheets.gd
```

图集缺失时游戏自动退回代码占位图形（提示词与网格对照见 `ART_PROMPTS.md`）。单位默认单帧立绘 + 丰富的**程序化动画**（按武器分攻击动作、两拍步态、受击/死亡），放入 `assets/anim/<key>_<state>.png` 帧带即自动改播逐帧动画。
